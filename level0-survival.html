⚠ the mèun trap

English builds big numbers as "ten-thousand." Thai has a dedicated word for 10,000 — หมื่น / mèun — so 25,000 baht is built as 2×10,000 + 5×1,000, not "twenty-five thousand."

How to drill this

  1. Get the numbers table cold. Drill baht amounts daily — convert real prices you see, e.g. 1,250 → nèung-pan-sǎwng-rói-hâa-sìp.
  2. Pick one template per section per day. Swap in five different nouns each time.
  3. Add kráp / kâ automatically until it stops feeling like an extra step.
